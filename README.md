# 🎬 Movie Search Engine

A responsive Movie Search Engine website built with HTML, CSS, and JavaScript. Users can search for movies, view posters, ratings, details, and watch trailers using the TMDB API.

## 🚀 Live Demo
🔗 https://rohanchoudhary6201.github.io/Movie-search-engine_/

"Open this link in Microsoft Edge. Chrome does not support the TMDB API"

> **Note:** The live demo currently works best in Microsoft Edge. I am working on improving compatibility with Google Chrome.

## ✨ Features
- 🔍 Search for movies
- 🎬 View movie details
- ⭐ Check ratings
- 🖼️ Display movie posters
- ▶️ Watch trailers
- 📱 Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- TMDB API
